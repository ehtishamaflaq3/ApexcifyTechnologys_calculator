# Apexcify Technologies Calculator

A modern and responsive **Calculator Web App** built using **HTML, CSS, and JavaScript**.  
This project demonstrates clean UI design, smooth animations, and full calculator functionality.
---

## Features

- Basic arithmetic operations (+, −, ×, ÷)
- Interactive buttons and display screen
- Real-time calculations
- Clear (C) functionality
- Keyboard support (Bonus)
- Smooth animations and hover effects
- Responsive design (works on mobile & desktop)
- Modern UI (Glassmorphism + Gradient Background)

---

## Technologies Used

- **HTML5** → Structure  
- **CSS3** → Styling & Animations  
- **JavaScript (Vanilla)** → Functionality

## How to Use

1. Enter numbers using buttons or keyboard  
2. Use operators (+, -, *, /)  
3. Press **= or Enter** to calculate  
4. Press **C** to clear screen  

---

## Code Highlights

- Used `eval()` for expression calculation
- Added `keydown` event for keyboard support
- Designed UI using **CSS Grid**
- Applied **hover & click animations**
